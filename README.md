## Introduction

Web application to solve the doubts of student. Student can post their question and other users can comment their solution. Student can find a teacher near their home using Google Map.

## Technology Stack:

1. Node.js
2. MongoDB
3. Bootstrap
4. HTML5
5. Javascript
